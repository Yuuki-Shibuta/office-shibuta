# しぶた行政書士事務所 HP改修プラン

## 目的
「青森市 行政書士」「青森 行政書士」など主要キーワードで**Google検索1位**を獲得する。

## 背景
- 現状の平均掲載順位：9.1位（2026年4月13日時点）
- 「青森市 行政書士」検索で現在**5位**（上位は競合4事務所）
- 上位競合はページ数が多い（15〜60ページ）。しぶた事務所は7〜8ページで劣勢
- コンテンツ量の差が順位差の主因

## 改修方針（5フェーズ）

### フェーズ1：業務詳細ページの大幅拡充【現在実施中】
**目的：** ページ数を増やし、キーワード網羅性を高める

既存の services.html（1ページに6サービス集約）を分割し、各サービスに独立した詳細ページを作成する。さらに業務を細分化して、関連キーワードごとにページを作る。

- `services/souzoku.html`（相続・遺言サポート）
- `services/syukatsu.html`（終活サポート）
- `services/kensetsu.html`（建設業許可申請）
- `services/kyoninka.html`（各種許認可申請）
- `services/keiyaku.html`（契約書作成・チェック）
- `services/houjin.html`（法人設立サポート）

**各ページの要件：**
- タイトルタグに「青森市」「青森」キーワードを含める
- メタディスクリプションに具体的な地域名と業務名
- 本文は1500〜3000字程度のボリューム
- 構造化データ（Service型、BreadcrumbList型）
- FAQを各ページに3〜5問配置
- 内部リンクで関連ページへ誘導

### フェーズ2：ブログ強化（月3〜4本ペース）
複合キーワード（「青森市 相続 費用」など）を狙った記事を継続投稿。

### フェーズ3：地域特化・実績ページ追加
青森市内エリア別ページ、お客様の声、料金表、アクセス詳細。

### フェーズ4：MEO（Googleマップ）強化
Googleビジネスプロフィールの定期投稿・口コミ獲得。

### フェーズ5：デザイン改修
電話ボタン固定表示、実績数字、顔写真など。

## フェーズ1の作業手順

1. `services/` ディレクトリを作成
2. 各サービス個別ページ（6ページ）を作成
3. services.html（ハブページ）を修正し、各個別ページへのリンクを設置
4. index.html のサービスカードのリンク先を各個別ページへ変更
5. sitemap.xml を更新
6. robots.txt を確認
7. ローカルで動作確認・リンク切れチェック
8. Git commit & push
9. GitHub Pages で反映確認
